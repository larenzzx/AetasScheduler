# 📅 AetasScheduler — Shift Scheduling Management System

AetasScheduler is a full-stack, rule-aware Operations Scheduling Portal. It replaces manual, error-prone spreadsheet-based planning with a modern, automated system for managing shift assignments, bi-weekly rotations, and emergency replacements.

The system features a reactive user interface built with Next.js, Supabase, and Tailwind CSS. It is backed by a robust, server-side scheduling validator and a conflict-resolving generator that ensures full compliance with labor configurations, rest guidelines, gender-safety companion rules, and mentorship requirements.
